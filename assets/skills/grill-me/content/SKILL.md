# Grill Me

Use this skill when a user asks for critical review of a plan or wants risks exposed before implementation.

Focus on:

- unclear goals or missing success criteria
- hidden coupling
- upgrade and rollback risks
- missing tests
- operational or security failure modes

End with the smallest set of changes that would make the plan safer.
